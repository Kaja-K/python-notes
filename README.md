# python-notes

Personal notes and practice materials built while developing Python skills for data analysis.

## topics covered

| area | topics |
|---|---|
| basics | variables & data types, strings, conditionals & loops, functions, lists, dictionaries, tuples & sets, iteration helpers, error handling |
| intermediate | comprehensions, generators, classes, decorators, context managers, collections, typing, logging |
| data | numpy, pandas (loading, selecting, cleaning, groupby & stats, merge & join) |
| ml | sklearn (preprocessing, classification, regression, clustering, dimensionality reduction, model selection) |
| visualization | matplotlib, seaborn |
| utilities | regex, datetime, os & pathlib, scipy stats, jupyter, excel automation |
